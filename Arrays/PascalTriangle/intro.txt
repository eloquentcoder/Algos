Given an integer numRows, return the first numRows of Pascal's triangle.

In Pascal's triangle, each number is the sum of the two numbers directly above it. Below is an example diagram
https://upload.wikimedia.org/wikipedia/commons/0/0d/PascalTriangleAnimated2.gif

DIFFICULTY LEVEL: Easy

Example 1:

Input: numRows = 5
Output: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
Example 2:

Input: numRows = 1
Output: [[1]]


Link to problem
https://leetcode.com/problems/pascals-triangle/

HINT: 

TIME_COMPLEXITY: 0(n**2) where n is the number of rows
SPACE_COMPLEXITY:  0(n**2) where n is the number of rows
